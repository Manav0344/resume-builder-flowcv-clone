# FlowCV Style Resume Builder

A production-ready React + Tailwind CSS resume builder with live split-pane editing, six templates, local persistence, drag-and-drop sections, JSON import/export, PDF export via jsPDF + html2canvas, themes, undo/redo, ATS and completeness scoring.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Create zip

```bash
npm run zip
```
